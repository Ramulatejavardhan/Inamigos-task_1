# InAmigos Awareness Website

A professional and responsive NGO awareness webpage created for the **InAmigos Foundation Internship Task**.

This project focuses on spreading awareness about the NGO’s mission, ongoing social initiatives, community development programs, and volunteer activities through a clean and modern web interface.

---

## 🌟 Project Objective

The goal of this project is to create an informative and visually appealing awareness webpage for **InAmigos Foundation** using only **HTML5** and **CSS3**.

The webpage highlights:

* NGO introduction
* Ongoing projects
* Social impact initiatives
* Volunteer activities
* Community welfare programs
* Call-to-action sections

---

## 🏢 About InAmigos Foundation

InAmigos Foundation is a Section 8 registered non-profit organization focused on:

* Education support
* Women empowerment
* Environmental sustainability
* Animal welfare
* Community welfare
* Youth skill development

The organization works through volunteer-driven initiatives and social campaigns to create positive community impact.
